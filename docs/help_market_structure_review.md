## Market Structure Review Help

Use this module to review how a selected asset is owned, supplied, and participated in across the market.

### Workflow

1. Select a dataset.
2. Select a focus asset.
3. Review ownership, float, supply, institutional, and index context.
4. Review mapped supply events.
5. Review the Structural Summary and Perspective Lenses.
6. Use Preview AI Export from Macro Interaction Tools when a structured bundle is required.

For custom datasets, upload both the market structure profile CSV and the supply events CSV.
