The **Asset Snapshot Scanner** module enables structured inspection and export of individual or grouped assets for diagnostic review, comparison, and downstream integration. It provides a clean snapshot of asset data, with selectable source layers and export-ready formatting.

Common uses include:

- Reviewing key statistics and structure of preloaded or user-uploaded assets
- Comparing asset characteristics across categories or custom universes
- Preparing cleaned, portable snapshots for use in other modules or workflows
- Generating .pkl and .csv exports for caching, modelling, or audit purposes

### Key Functions

- **Snapshot Source Selection:** Choose from default system assets or user-defined sets
- **Descriptive Summary Panel:** View structural metadata, key stats, and data coverage
- **Export Utilities:** Save cleaned snapshot as .pkl or .csv to user-defined location
- **Format-Ready Outputs:** Ensures alignment with downstream analytical modules
